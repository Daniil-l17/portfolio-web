## Запускается локально

Ссылка на сайт

```bash

```

Клонировать проект

```bash
https://github.com/Daniil-l17/portfolio-web.git
```

Устанавливать зависимости

```bash
yarn install
```

Запустите приложение в dev

```bash
yarn dev
```

Запустите приложение в рабочем режиме

```bash
yarn build
```

Затем

```bash
yarn start
```
